Допустимые методы для адреса /get_classifier_result/<version> GET и POST

Читает полученный json. Если version==1, возвращает json с версией и полем predict из входящего json {"version":1, "predict":<predict_value>}

Если version==0, то возвращает json с версией и полем old_predict из входящего json {"version":0, "predict":<old_predict_value>}

Если вам понравилось задонатьте на спасение морских котиков(посмотрите на этих милашек)
